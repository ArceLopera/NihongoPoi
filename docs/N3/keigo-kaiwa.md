## Offering An Invitation


![Invite](../Images/invite.jpg)



A:部長、ちょっとよろしいでしょうか。</br>
B:何？</br>
A:再来週の日曜日、なにかご予定がありますか。</br>
B:日曜日・・・・どうして？何かあるの？</br>
A:もうすぐ海外に転勤する人がいるので、送別会をしようと思っているんです。</br>部長もご一緒にいかがかと思いまして・・・。</br>
B:おお、いいね！行きたいな。お菓子か何か買っていったほうがいい？</br>
A:いえいえ、お気遣いなく。では、またご連絡します。楽しみにしております。</br>


### 例文

1. Address someone：声をかける

    部長、すみません。</br>
    いま、よろしいですか。・ちょっとよろしいでしょうか。</br>
    今、お時間いただけますか。</br>

2. Ask about someone's plans
    
    日曜日は、なにかご予定がありますか。</br>
    来週の火曜日、小林さんのご都合はいかがでしょうか。

3. Offer an Invitation
    
    + 部長にもぜひ来ていただきたいんですが・・・。（強く誘うときはこの表現を使い）</br>
    + 部長にもご出席いただけないかとおもいまして。</br>
    「〜まして」は「〜ます」のて形。「〜ので」と同じ意味。書くときは使いません。</br>
    + よろしかたら、部長もいらっしゃいませんか。</br>
    + よろしければ、部長もいかがでしょうか。</br>
    + もしご都合がよろしければ、部長もいかがかとおもいまして・・・。</br>

4. 「ご遠慮無く」　　「お・ご〜なく」＝「〜ないでください」
    
    「ご心配なく」「お気遣いなく」「おかまいなく」など <s>「ご心配なくてね」</s></br>
    A:私もパーティーに行っていいでしょうか。</br>
    B:もちろん。どうぞご遠慮無く。</br>
    A:飲み物などを買っていきましょうか。</br>
    B:いえいえ。私が用意しますから、お気遣いなく。</br>

    すべての同士には使えないので注意！
    <s>「お書きなく」</s>　　<s>「お話しなく」</s>

5. 「〜ております」　　＝「〜ています」の謙譲語
    
    お返事をお待ちしております。</br>
    当日お会いできるのを楽しみにしております。</br>
    大学で日本映画を研究しております。 </br>

6. 「〜でしょうか」
   
    コーヒーはお好きでしょうか。</br>
    明日はお忙しいでしょうか。</br>
    ご都合はいかがでしょうか。</br>
    いつがよろしいでしょうか。</br>

7. 「〜ていただきたいんですが」
   
    先生にも来ていただきたいんですが・・・。</br>
    ぜひ、パーティーに出席していただきたいんですが・・・。</br>
    先生にも、是非参加していただきたいんですが・・・。</br>

8. 「〜もいかがかとおもいまして」
   
    この映画、ご一緒に田中さんもいかがかと思いまして・・・。
